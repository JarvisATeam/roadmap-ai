# OFFER_AI_OPS_SNAPSHOT_48H

## Formål
Produktisere GinieSystem sin første salgbare leveranse som en smal, rask og bevisbar operasjon.

## Navn
AI Ops Snapshot — 48t

## Hva kunden kjøper
En komprimert gjennomgang av eksisterende AI-/automatiseringsrigg med fokus på:
- hva som faktisk kjører
- hva som er bevist
- hva som er uklart
- hva som blokkerer inntekt
- hva som bør gjøres neste

## Passer for
- små selskaper med AI-verktøy, bots eller automasjoner i drift
- gründere med kaotisk repo/runtime-bilde
- team som mangler én autoritativ sannhetskilde
- systemer med flere noder, APIer, dashboards eller agenter

## Input fra kunde
- repo-tilgang eller eksport
- tilgang til runtime-node eller health-proof
- liste over aktive tjenester/domener
- ønsket mål:
  - stabil drift
  - raskere levering
  - første inntekt
  - kostkutt
  - opprydding

## Output til kunde
- truth-source dokument
- deploy map
- auth-/API-proof der mulig
- kost-baseline
- node role map
- topprangerte flaskehalser
- konkret 7-dagers handlingsplan
- quick wins med lav friksjon

## Proof-standard
- Git Proof Pack der repo kan verifiseres
- runtime-proof
- endpoint-proof
- auth-proof hvis tilgjengelig
- eksplisitt merking:
  - VERIFISERT
  - UKJENT
  - MÅ INVENTERES

## Leveranseformat
- Markdown-dokumenter
- Notion-klare seksjoner
- én samlet handover
- null fluff

## Ikke inkludert
- full rebuild
- store arkitekturendringer uten proof
- destruktiv opprydding uten eksplisitt godkjenning
- hemmeligheter/nøkler i leveranse

## Kundeverdi
- mindre kaos
- raskere beslutninger
- mindre feilsøking
- enklere salg/investor/demo
- klar vei mot første krone eller bedre drift

## Operativ flyt
1. inventar
2. proof
3. mapping
4. flaskehalsanalyse
5. 7-dagers plan
6. handover

## Pris
- AVKLARES KUN MED CLEAR
